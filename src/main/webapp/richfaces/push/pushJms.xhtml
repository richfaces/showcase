This example introduces Push via JMS capability - messages from Java Messaging System are observed by RichFaces Push and transported via Comet to client.

For the purpose of observing events from the server, client creates pernament connection to the server either by WebSocket connection or long-polling method. Pernament connections are managed by Atmosphere Framework.

In this example Message payload is serialized to JSON and transfered to the client. On the client-side, dataavailable event is triggered with event.rf.data payload which contains serialized data.
